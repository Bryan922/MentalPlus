Configuration Supabase

Configurez votre base de données Supabase pour MentalSerenity

1 Créer un Projet Supabase

Si vous n'avez pas encore de projet Supabase, suivez ces étapes :

  1. Allez sur supabase.com
  2. Créez un compte ou connectez-vous
  3. Cliquez sur "New Project"
  4. Remplissez les informations :
    • Name : mentalserenity
    • Database Password : Créez un mot de passe fort
    • Region : Choisissez la région la plus proche
  5. Cliquez sur "Create new project"

2 Récupérer les Clés d'API

Dans votre projet Supabase :

  1. Allez dans Settings > API
  2. Copiez les informations suivantes :
L'URL de votre projet Supabase
La clé publique anonyme

3 Déployer le Schéma de Base de Données

Dans votre projet Supabase :

  1. Allez dans SQL Editor
  2. Cliquez sur "New query"
  3. Copiez le contenu du fichier database-schema-supabase.sql
  4. Collez dans l'éditeur et cliquez sur "Run"
